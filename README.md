## Real Estate Data Analysis

This workbook explores how property characteristics affect residential sale prices. The dashboard focuses on features such as property quality, lot size, living area, basement condition, and neighborhood.

Skills demonstrated:

- Tableau dashboard design
- Feature comparison
- Price trend exploration
- Visual storytelling for real estate data

## Bank Marketing Campaign Analysis

This workbook analyzes customer demographics and campaign activity to understand what factors are connected to term deposit subscription behavior.

Skills demonstrated:

- Customer segmentation
- Campaign performance analysis
- KPI reporting
- Filterable dashboard design

## Sentiment-Aware Restaurant Recommender

This workbook visualizes output from a machine learning workflow that used review sentiment, embeddings, clustering, and recommendation scores.

Skills demonstrated:

- Machine learning result interpretation
- Sentiment analysis visualization
- Cluster-based exploration
- Recommendation ranking presentation

## Tools

- Tableau
- Excel
- Python for supporting machine learning and preprocessing workflows
- GitHub for version control and documentation

## Purpose

These projects were created to strengthen practical analytics skills: asking useful business questions, preparing data for analysis, building dashboards, and explaining insights clearly to non-technical audiences.
